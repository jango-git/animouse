What's happening here?
This example demonstrates PolarBlendTree - a 2D animation blending system using polar coordinates for intuitive directional movement control.
• Polar Control: Semicircle area for natural movement
• Direction & Speed: Azimuth sets direction, radius sets intensity
• Multi-layer Blending: Smooth transitions between all states
• Visual Feedback: Real-time coordinate display
Perfect for character locomotion, vehicle controls, or any scenario requiring directional movement with variable intensity.
