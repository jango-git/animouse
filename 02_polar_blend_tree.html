What's happening here?
This example demonstrates PolarBlendTree - a 2D animation blending system using polar coordinates (angle + distance from center).
Move the cursor in the circular area to control:
• Azimuth: Direction (left/forward/right)
• Radius: Intensity (idle/walk/run)
• Center = idle, inner ring = walk, outer ring = run
Perfect for omnidirectional movement systems, vehicle controls, or any scenario requiring natural 2D navigation with varying intensity.
