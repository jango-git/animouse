What's happening here?
This example demonstrates Event-Driven Transitions and Automatic Transitions - the state machine responds to events and automatically returns to previous states.
• Speed Slider: Controls LinearBlendTree (idle→walk→run)
• Jump Button: Triggers event transition to jump
• Auto Return: Jump automatically returns to movement
• Different jump animations based on current speed
Perfect for action games, interactive characters, or any system requiring temporary state changes with automatic cleanup.
