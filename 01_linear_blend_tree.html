What's happening here?
This example demonstrates LinearBlendTree - a 1D animation blending system that smoothly transitions between multiple animations based on a single parameter.
• 1D Blending: Single slider controls 5 animations
• Smooth Transitions: Natural movement between states
• Value Mapping: -1 to 1 range covers all animations
• Real-time Control: Instant response to input
Perfect for locomotion systems, weapon handling, or any scenario where you need smooth transitions along a single axis of control.
